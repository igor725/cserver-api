# Color

## Поля

### int: r

```lua
color.r
```

### int: g

```lua
color.g
```

### int: b

```lua
color.b
```

## Функции

### color:set

```lua
color:set(int: R, int: G, int: B)
```

```lua
color:set(int: R, int: G, int: B, int: A)
```

Устанавливает объекту Color указанные компоненты.

### color:get

```lua
color:get()
```

Возвращает текущие значение цвета.
