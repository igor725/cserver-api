# setInfo

Updates info about script.

## Usage

```lua
setInfo(table: info)
```


## Info Table
| Field     | Type      | Description                                         |
|-----------|-----------|-----------------------------------------------------|
| version   | int       | Script version.                                     |
| home      | string    | Script home page (URL).                             |
| hotreload | boolean   | Allow/disallow script reloading via reload command. |
