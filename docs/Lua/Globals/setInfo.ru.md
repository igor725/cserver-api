# setInfo

Обновляет информацию о скрипте.

## Usage

```lua
setInfo(table: info)
```


## Info Table
| Field     | Type      | Description                                         |
|-----------|-----------|-----------------------------------------------------|
| version   | int       | Версия скрипта.                                     |
| home      | string    | Домашняя страница скрипта (URL).                    |
| hotreload | boolean   | Разрешает/запрещает горячую перезагрузку скрипта.   |
